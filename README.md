## Shell-Based Digital Library System (Team Project)

A command-line digital library system developed using Bash scripting.

The system supports:
- User authentication
- Book browsing and search
- Admin management (add/delete books, change password)

**Technologies:** Bash, Linux (Ubuntu), Git, GitHub
